## Descripción del Proyecto

Desarrollé de forma independiente una plataforma de comercio electrónico especializada en la venta de celulares y accesorios. El sistema permite a los usuarios explorar productos, gestionar pedidos y comunicarse directamente mediante WhatsApp para finalizar sus compras.

Participé en todas las etapas del desarrollo, incluyendo el diseño de la interfaz, arquitectura de la aplicación, modelado de la base de datos, implementación del backend, autenticación de usuarios, seguridad y despliegue de la plataforma.

## Funcionalidades Principales

- Catálogo dinámico de productos.
- Gestión de marcas, categorías y colores.
- Sistema de autenticación y registro de usuarios.
- Roles de administrador y cliente.
- Gestión de pedidos con validación del lado del servidor.
- Integración con WhatsApp para el envío de pedidos.
- Administración de imágenes de productos en la nube.
- Diseño responsive para dispositivos móviles y escritorio.
- Implementación de medidas de seguridad para protección de datos y control de acceso.

## Tecnologías Utilizadas

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router

### Backend
- Edge Functions (Deno + TypeScript)
- PostgreSQL
- Sistema de autenticación y autorización
- Row Level Security (RLS)
